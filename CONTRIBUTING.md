# Contributing

Thanks for helping improve **Awesome Cyber & AI Arsenal**! 🙌

## How to add a tool

1. **Fork** this repository and create a branch.
2. Add your tool to the **most appropriate category** in `README.md`.
   - Not sure if it's offensive or defensive? Offensive = used to *find/exploit*, Defensive = used to *detect/protect*. If it's dual-use, place it where it's most commonly used and note it in the description.
   - AI/LLM tools go under **🤖 AI / LLM Security**.
3. Use this exact row format:

   ```markdown
   | [tool-name](https://link) | One factual sentence describing what it does. |
   ```

4. Keep descriptions **short, neutral and accurate** — no marketing language.
5. Make sure the **link works** and points to the official source (GitHub repo or homepage).
6. Open a **Pull Request** with a clear title, e.g. `add: <tool-name> to <category>`.

## Rules

- One tool per line.
- No duplicates — search the README first.
- No tools whose *sole purpose* is illegal activity with no legitimate security use.
- New categories are welcome when a tool genuinely doesn't fit an existing one.

## Reporting issues

Found a dead link, a miscategorized tool, or a wrong description? Open an **Issue** and we'll fix it.
